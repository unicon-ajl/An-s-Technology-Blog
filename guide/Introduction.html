关于小安的技术博客 ✨ ​

你好，我是 小安 —— 一名热爱前端开发的工程师。
这里是我记录思考、沉淀经验、分享成长的技术角落。

👨‍💻 博主简介 ​

  • 从业方向：前端开发（主攻 Vue / React 技术栈）
  • 擅长领域：Vue 生态开发、高复用组件设计、工程化提效
  • 学习理念：以输出倒逼输入，用实践验证理论，在迭代中持续精进

📚 博客定位 ​

本博客聚焦 现代前端开发 的实战与原理，内容涵盖：

  • 🧠 框架深度：Vue / React 核心机制、性能优化与最佳实践
  • ⚙️ 工程化体系：Vite / Webpack 配置、构建优化、CI/CD 流程
  • 🛠️ 实战项目：VitePress 建站、组件库开发、工具脚手架搭建
  • 🕳️ 踩坑复盘：真实开发中遇到的问题、排查思路与解决方案

写作初衷：不堆砌概念，只讲 有用、能跑、可复用 的代码与经验。

📝 面经模块 ​

专为前端求职者打造的面试指南，内容包括：

  • 🔥 高频面试题精讲（JavaScript / CSS / Vue / React / 浏览器原理等）
  • 💡 答题思路拆解 + 代码示例 + 延伸思考
  • 📈 从“答得出”到“答得漂亮”的进阶路径
