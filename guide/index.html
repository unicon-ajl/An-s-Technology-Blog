📘 使用指南 ​

欢迎阅读本技术博客的使用说明！
无论你是初次访问，还是常驻读者，这份指南都能帮你更高效地浏览与使用本站内容。

1. 网站功能说明 ​

  • 导航栏（顶部右侧）
    快速切换主要模块，如「首页」「指南」「前端」「...」等。

  • 侧边栏（左侧）
    展示当前目录下的文章列表，支持展开/收起，便于快速定位章节。

  • 全文搜索（顶部左侧）
    支持本地模糊搜索，输入关键词即可实时匹配文章标题与内容。

  • 主题切换（右上角）
    点击右上角 🌞/🌙 图标，一键切换日间 / 夜间模式，保护视力更舒适。

  • 回到顶部（右下角）
    页面向下滚动超过 300px 时，右下角将显示「↑」按钮，点击即可平滑返回顶部。

2. 内容分类概览 ​

本博客内容按技术领域组织，主要包含以下模块：

🧱 前端基础与框架 ​

  • HTML / CSS / JavaScript（ES6+）
  • Vue 2 & Vue 3（响应式原理、Composition API、生态工具） & React
  • 微信小程序 / 跨端开发（UniApp）
  • 组件库设计与封装实践

⚙️ 工程化与构建工具 ​

  • Vite（极速开发体验）

  • 包管理（npm / Yarn / pnpm）
  • 代码规范（ESLint + Prettier + Git Hooks）

🧰 实用工具与技巧 ​

  • 通用工具函数（utils）：防抖节流、深拷贝、类型判断等
  • 开发效率工具：VS Code 插件推荐、WebStorm 配置技巧
  • 调试与性能分析：Chrome DevTools 高级用法

💡 所有内容均源于真实项目经验，注重 可运行、可复用、可理解。
