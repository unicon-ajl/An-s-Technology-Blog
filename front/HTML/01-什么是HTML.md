
# 什么是 HTML？

> 📌 **HTML（HyperText Markup Language）**，即“超文本标记语言”，是用于创建网页结构的标准标记语言。

它不是编程语言，而是一种**描述性语言**——你用它告诉浏览器：“这里是一段标题，那里是一个列表，这个是图片……”

---

## 一个最简单的 HTML 页面

```html
<!DOCTYPE html>
<html>
<head>
  <title>我的第一个页面</title>
</head>
<body>
  <h1>你好，世界！</h1>
  <p>这是由 HTML 构建的网页。</p>
</body>
</html>
```

当你在浏览器中打开这个文件，会看到：

```
你好，世界！
这是由 HTML 构建的网页。
```

> 💡 浏览器读取 HTML 文件后，将其**解析为 DOM（文档对象模型）**，再结合 CSS 和 JavaScript 渲染成你看到的页面。

---

## 核心概念

### 1. **标签（Tag）**
- 用尖括号 `<>` 包裹，如 `<p>`、`<img>`；
- 大多数标签成对出现：**开始标签** + **结束标签**  
  ```html
  <p>这是一个段落。</p>
  ```
- 少数自闭合标签（无需结束）：  
  ```html
  <img src="cat.jpg" alt="一只猫">
  <br>
  ```

### 2. **元素（Element）**
- 标签 + 内容 = 元素  
  ```html
  <h1>前端技术全景</h1>  ← 这是一个 h1 元素
  ```

### 3. **属性（Attribute）**
- 提供额外信息，写在开始标签内：  
  ```html
  <a href="https://example.com">链接</a>
  <img src="photo.jpg" alt="风景照">
  ```
  - `href`、`src`、`alt` 都是属性；
  - 属性值**必须用引号包裹**（推荐双引号）。

## HTML 能做什么？

| 功能 | 示例标签 |
|------|--------|
| 定义标题 | `<h1>`–`<h6>` |
| 段落文本 | `<p>` |
| 插入图片 | `<img>` |
| 创建链接 | `<a>` |
| 列表 | `<ul>`, `<ol>` |
| 表单输入 | `<input>`, `<button>` |
| 语义化结构 | `<header>`, `<main>`, `<article>` |

> ⚠️ HTML **不能**：
> - 改变颜色或布局（那是 CSS 的工作）；
> - 实现交互逻辑（那是 JavaScript 的工作）。

---

## 常见误区

❌ “HTML 是过时的技术”  
✅ 错！它是 Web 的基石，永远不可或缺。

❌ “只要页面能显示就行，标签随便用”  
✅ 错！滥用 `<div>` 会导致 SEO 差、无障碍体验差、代码难维护。
