
# HTML 与 SEO

> 🔍 **SEO（搜索引擎优化）从 HTML 开始**。  
> 搜索引擎首先“阅读”的就是你的 HTML 结构——写得好，自然排名高。

---

## 为什么 HTML 影响 SEO？

搜索引擎爬虫（如 Googlebot）通过解析 HTML 来：
- 理解页面主题；
- 提取标题与关键内容；
- 判断内容质量与相关性。

> ✅ 良好 HTML = 更准确的内容理解 = 更高的搜索排名。

---

## 关键 HTML 元素与 SEO

### 1. `<title>` —— **最重要的 SEO 标签**
```html
<title>HTML 详细剖析 | 小安的技术博客</title>
```
- 显示在搜索结果**第一行**；
- 建议长度：**50–60 字符**；
- 包含核心关键词（如 “HTML 教程”）；
- 每页唯一，避免重复。

> 💡 格式建议：`主关键词 | 网站名`

---

### 2. `<meta name="description">` —— 搜索摘要
```html
<meta name="description" content="系统学习 HTML 语义化、表单、无障碍等核心知识，适合前端初学者。">
```
- 显示在搜索结果**第二段**（非绝对，但常被采用）；
- 建议长度：**150–160 字符**；
- 清晰描述页面价值，吸引点击。

> ⚠️ 注意：`description` **不影响排名**，但影响点击率（CTR）。

---

### 3. 标题层级 `<h1>`–`<h6>`
- `<h1>`：页面主标题，**包含核心关键词**，每页仅一个；
- `<h2>`–`<h3>`：章节标题，形成内容大纲；
- 避免跳级（如 `h1 → h3`），保持逻辑清晰。

> ✅ 示例：
> ```html
> <h1>HTML 表单最佳实践</h1>
> <h2>如何正确使用 label</h2>
> <h2>表单验证与用户体验</h2>
> ```

---

### 4. 语义化结构
搜索引擎能识别 `<article>`、`<main>`、`<nav>` 等标签：
- `<main>` 内容被视为**核心信息**；
- `<aside>`、`<footer>` 被视为**辅助内容**；
- 语义清晰的页面更容易被正确索引。

> 📌 提示：避免将广告或无关内容放在 `<main>` 中。

---

### 5. 图像 `alt` 文本
```html
<img src="html-diagram.png" alt="HTML 文档结构示意图：包含 head、body、header 等标签">
```
- 帮助搜索引擎理解图片内容；
- 图片搜索的重要依据；
- 关键词自然融入，**切勿堆砌**。

---

### 6. 内部链接 `<a>`
- 使用**描述性锚文本**：
  ```html
  <!-- ❌ 差 -->
  <a href="/html">点击这里</a>

  <!-- ✅ 好 -->
  <a href="/html">HTML 语义化完整指南</a>
  ```
- 合理链接相关页面，提升网站结构权重。

---

## 其他 SEO 友好实践

| 实践 | 说明 |
|------|------|
| **规范 URL** | 使用 `<link rel="canonical" href="...">` 防止重复内容 |
| **结构化数据** | 添加 JSON-LD（如 Article、Breadcrumb）帮助富媒体展示 |
| **移动端适配** | `<meta name="viewport" content="width=device-width">` |
| **快速加载** | 精简 HTML，懒加载图片/iframe |

> 💡 结构化数据示例（文章）：
> ```html
> <script type="application/ld+json">
> {
>   "@context": "https://schema.org",
>   "@type": "Article",
>   "headline": "HTML 详细剖析",
>   "description": "系统学习 HTML 核心知识...",
>   "author": { "@type": "Person", "name": "小安" }
> }
> </script>
> ```

---

## 常见 SEO 误区

❌ 堆砌关键词（如 `<h1>HTML HTML 教程 HTML 入门</h1>`）→ 被判定为作弊  
❌ 隐藏文字（如白色文字 on 白底）→ 违反搜索引擎政策  
❌ 大量重复 `<title>` 或空 `description` → 降低页面价值评分  

---

## 总结：HTML 是 SEO 的地基

| 元素 | SEO 作用 |
|------|--------|
| `<title>` | 决定搜索结果标题 + 排名权重 |
| `<meta description>` | 影响点击率 |
| `<h1>` | 定义页面主题 |
| 语义化标签 | 帮助爬虫理解内容结构 |
| `alt` + 内部链接 | 提升内容关联性与可发现性 |

> ✅ **好 HTML = 好内容 + 好结构 + 好语义 = 自然 SEO**
