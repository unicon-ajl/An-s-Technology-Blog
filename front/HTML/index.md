## HTML 详细剖析

> 🌐 **HTML（HyperText Markup Language）是 Web 的骨架**。  
> 它不负责样式，也不处理逻辑，只专注一件事：**描述内容的结构与含义**。

本系列将带你从零开始，系统掌握 HTML 的语法、语义、最佳实践与现代用法。无论你是初学者，还是想夯实基础的开发者，都能从中受益。

---

## 为什么学 HTML？

- 所有网页都由 HTML 构成；
- 是 CSS 和 JavaScript 的载体；
- 良好的 HTML = 更好的 SEO + 更强的可访问性（无障碍）+ 更易维护的代码。

---

## 学习目标

✅ 掌握 HTML 基础语法  
✅ 理解语义化标签的意义  
✅ 能写出结构清晰、SEO 友好、屏幕阅读器兼容的页面  
✅ 避开常见误区与过时写法
