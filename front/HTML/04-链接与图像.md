
# 链接与图像

> 🔗🖼️ **链接（`<a>`）让网页互联，图像（`<img>`）让内容生动**。  
> 它们是 HTML 中最常用的两个元素，也是 SEO 与无障碍的关键。

---

## 链接：`<a>` 标签

### 基本用法
```html
<a href="https://example.com">访问示例网站</a>
```

- `href` 属性指定目标地址；
- 点击后跳转到新页面或锚点。

---

### 常见链接类型

| 类型 | 示例 | 说明 |
|------|------|------|
| **外部链接** | `<a href="https://vuejs.org">Vue 官网</a>` | 跳转到其他网站 |
| **内部链接** | `<a href="/guide/html">HTML 指南</a>` | 站内跳转 |
| **锚点链接** | `<a href="#section1">跳转到章节</a>` | 页面内滚动 |
| **邮箱链接** | `<a href="mailto:me@example.com">发邮件</a>` | 打开邮件客户端 |
| **下载链接** | `<a href="file.pdf" download>下载 PDF</a>` | 强制下载 |

> 💡 提示：站内链接建议使用**相对路径**（如 `/guide/`），便于部署。

---

### 无障碍要点
- **链接文字要有意义**：
  ```html
  <!-- ❌ 差：用户不知道去哪 -->
  <a href="/report.pdf">点击这里</a>

  <!-- ✅ 好：清晰说明目标 -->
  <a href="/report.pdf">2025 年技术报告（PDF）</a>
  ```
- 外部链接可加图标提示（需 ARIA 支持）：
  ```html
  <a href="https://github.com" target="_blank" aria-label="在新窗口打开 GitHub">
    GitHub
  </a>
  ```

> ⚠️ 避免 `target="_blank"` 不加 `rel="noopener"`（安全风险）：
> ```html
> <a href="..." target="_blank" rel="noopener">外部链接</a>
> ```

---

## 图像：`<img>` 标签

### 基本语法
```html
<img src="photo.jpg" alt="一只在草地上奔跑的金毛犬">
```

- `src`：图像路径（相对或绝对）；
- **`alt`：替代文本（必填！）**

---

### 为什么 `alt` 如此重要？

| 场景 | 作用 |
|------|------|
| **图片加载失败** | 显示文字描述 |
| **屏幕阅读器** | 朗读给视障用户听 |
| **SEO** | 帮助搜索引擎理解图片内容 |

> ✅ 好 `alt`：简洁、准确、有上下文  
> ❌ 坏 `alt`：`alt="图片"`、`alt=""`（除非纯装饰）

### 装饰性图片处理
如果图片仅用于美化（无信息价值），可设空 `alt`：
```html
<img src="decoration-border.png" alt="">
```
> 📌 这样屏幕阅读器会**跳过**它。

---

## 组合用法：图文语义化

使用 `<figure>` 和 `<figcaption>` 包裹带说明的图片：

```html
<figure>
  <img src="chart.png" alt="2025 年前端框架使用趋势">
  <figcaption>图：主流前端框架市场份额对比</figcaption>
</figure>
```

- `<figure>` 表示独立内容单元；
- `<figcaption>` 是其标题或说明；
- 适合图表、照片、代码块等。

---

## 响应式图像（进阶）

通过 `srcset` 和 `sizes` 适配不同设备：

```html
<img
  src="small.jpg"
  srcset="medium.jpg 1000w, large.jpg 2000w"
  sizes="(max-width: 600px) 100vw, 50vw"
  alt="响应式风景图"
>
```

> 💡 初学者可先用单一 `src`，后续再优化性能。

---

## 总结

| 元素 | 关键属性 | 必做事项 |
|------|--------|--------|
| `<a>` | `href` | 链接文字有意义，外部链接加 `rel="noopener"` |
| `<img>` | `src`, `alt` | **`alt` 必填**，装饰图设为空 |

> ✅ 好链接 + 好图片 = 可访问、可搜索、用户体验佳的网页。
