
# 颜色与背景

> 🎨 **颜色传递情绪，背景定义空间**。  
> 掌握现代 CSS 的色彩表达与背景控制，让你的设计既精准又富有表现力。

---

## 颜色表示法

CSS 支持多种方式定义颜色，各有适用场景：

### 1. 关键字（Named Colors）
```css
color: red;
background: navy;
```
- 简单直观，但只有 140+ 个标准色（如 `tomato`, `rebeccapurple`）；
- 适合快速原型，不适合精确设计。

---

### 2. 十六进制（HEX）
```css
color: #ff5733;     /* 六位 */
color: #f53;        /* 三位简写（等价于 #ff5533） */
```
- 最广泛支持；
- 无法直接调整透明度（需用 `rgba` 或 `hsla`）。

---

### 3. RGB / RGBA
```css
color: rgb(255, 87, 51);
color: rgba(255, 87, 51, 0.8); /* 第4个参数：透明度 0~1 */
```
- 适合程序化生成颜色（如 JS 动态计算）；
- `rgba` 可实现半透明叠加效果。

---

### 4. HSL / HSLA ✅ **推荐用于设计系统**
```css
color: hsl(12, 100%, 60%);       /* 色相/饱和度/亮度 */
color: hsla(12, 100%, 60%, 0.9); /* 带透明度 */
```
- **更符合人类直觉**：
  - 色相（Hue）：0–360°（红→绿→蓝→红）；
  - 饱和度（Saturation）：0%（灰）→ 100%（纯色）；
  - 亮度（Lightness）：0%（黑）→ 50%（标准）→ 100%（白）。
- 易于生成同色系变体（如 hover 状态调亮 10%）。

> 💡 示例：按钮主色 + 悬停色
> ```css
> .btn {
>   background: hsl(210, 100%, 50%); /* 蓝色 */
> }
> .btn:hover {
>   background: hsl(210, 100%, 40%); /* 更暗的蓝 */
> }
> ```

## 背景（Background）控制

CSS 提供一组 `background-*` 属性精细控制背景：

### 1. `background-color`
```css
header { background-color: #f8f9fa; }
```

### 2. `background-image`
```css
.hero { background-image: url("banner.jpg"); }
```

### 3. `background-repeat`
- `repeat`（默认）、`no-repeat`、`repeat-x`、`repeat-y`

### 4. `background-position`
定位背景图起点：
```css
background-position: center;      /* 居中 */
background-position: top right;   /* 右上角 */
background-position: 20px 50px;   /* 精确偏移 */
```

### 5. `background-size`
控制背景图尺寸：
```css
background-size: cover;    /* 覆盖整个容器（常用） */
background-size: contain;  /* 完整显示，可能留空 */
background-size: 100px 50px; /* 固定尺寸 */
```

### 6. 简写语法（推荐）
```css
.banner {
  background: #eee url("pattern.png") no-repeat center/cover;
}
```
顺序：`[color] [image] [repeat] [position] / [size]`

> ⚠️ 注意：`position / size` 中的 `/` 是固定语法，不能省略。

## 渐变（Gradients）

CSS 原生支持无需图片的渐变背景：

### 1. 线性渐变（Linear Gradient）
```css
.gradient {
  background: linear-gradient(to right, red, yellow);
  background: linear-gradient(45deg, #ff9a9e, #fad0c4);
}
```
- 方向：`to right`、`to bottom left`、角度（`90deg` = 向下）；
- 可多色：`linear-gradient(red, orange, yellow, green)`

### 2. 径向渐变（Radial Gradient）
```css
.radial {
  background: radial-gradient(circle, #fff, #ccc);
}
```
- 形状：`circle`（圆形）或 `ellipse`（椭圆，默认）；
- 可指定中心点和尺寸。

> 💡 实战：按钮悬停微渐变
> ```css
> .btn {
>   background: linear-gradient(to bottom, #007bff, #0069d9);
> }
> ```

## 多重背景

一个元素可叠加多个背景图/渐变：
```css
.multi-bg {
  background:
    url("icon.png") no-repeat top left,
    linear-gradient(to right, #eee, #fff),
    #f0f0f0;
}
```
- **从上到下绘制**，第一项在最顶层；
- 最后一项通常是纯色兜底。

## 可访问性与性能提示

✅ **确保文本与背景对比度 ≥ 4.5:1**（尤其浅灰文字 on 白底）；  
✅ **避免纯色大面积闪烁**（可能引发光敏性癫痫）；  
✅ **大图背景使用 `loading="lazy"` 的 `<img>` 替代？不！**  
 → 背景图不会被搜索引擎索引，重要内容请用 `<img>` + `alt`；  
✅ **渐变比图片轻量**，优先用 CSS 渐变替代 PNG 背景。

---

✅ 本篇亮点：
- 推荐 HSL 作为设计友好型颜色模型；
- 背景简写语法清晰标注格式（含 `/`）；
- 渐变示例贴近 UI 实际需求（按钮、横幅）；
- 强调“背景图 ≠ 内容图”的无障碍原则。
